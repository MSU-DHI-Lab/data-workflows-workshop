# Day 03 Glossary

- **Cleaning**: changing data values to fix issues (trim, normalize, correct).
- **Validation**: checking data against rules to confirm it meets expectations.
- **Quality check**: a set of checks data should pass before you treat it as ready to share or publish.
- **Validation rules**: the rules for fields, types, and allowed values.
- **Quiet failure**: a data error that passes silently through a flow.
- **Pandera**: a Python library for declarative data validation on pandas dataframes.
- **Check**: a Pandera rule applied to a column or the whole dataframe.
