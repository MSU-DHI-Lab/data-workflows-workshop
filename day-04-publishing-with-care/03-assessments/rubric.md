# Rubric

| Criterion | Emerging | Proficient | Strong |
| --- | --- | --- | --- |
| Package completeness | Missing docs or checks | Data, README, dictionary, notes on what changed present | Package thorough, organized, clearly labeled |
| Licensing & citation | License unclear; citation incomplete | LICENSE chosen; CITATION.cff filled | Clear license rationale; citation includes DOI/version when available |
| Metadata quality | .zenodo.json incomplete | Required fields filled; valid .zenodo.json file | Metadata clear, keywords useful, aligned with README |
| Checklist use | Checklist ignored | Checklist mostly complete | Checklist complete and matches the package files |
| Practice/live path | Path unclear | Chosen path executed; deliverables saved | Path executed with clear notes; DOI recorded if live |
| Colleague handoff | Not considered | Streamlit or equivalent noted | Streamlit app or plan demonstrates thoughtful handoff |
